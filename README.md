# Safety-course-
# 🛡️ ملخص مادة الصحة والسلامة المهنية

## 📌 1. المفاهيم الأساسية والتعاريف
* **الصحة المهنية:** (اكتب هنا التعريف باختصار شديد، مثلاً: الحفاظ على سلامة العاملين البدنية والنفسية).
* **المخاطر (Hazards):** أي مصدر محتمل للضرر أو الأذى.
* **الخطر/الاحتمالية (Risk):** احتمالية حدوث الضرر نتيجة التعرض للمخاطر.

## ⚠️ 2. تصنيف المخاطر المهنية
| نوع الخطر | أمثلة عليه | طرق الوقاية والسيطرة |
| :--- | :--- | :--- |
| **مخاطر فيزيائية** | الضوضاء، الحرارة، الإشعاع | استخدام سدادات الأذن، العزل |
| **مخاطر كيميائية** | الغازات، الأبخرة، الأحماض | التهوية، الكمامات الخاصة |
| **مخاطر بيولوجية** | الفيروسات، البكتيريا، الفطريات | التعقيم، التطعيمات |
| **مخاطر ميكانيكية** | الآلات الحادة، السقوط | الحواجز الواقية، أحزمة الأمان |
| **مخاطر إيرغونومية** | الجلوس الخاطئ، رفع أحمال ثقيلة | كراسي مريحة، تدريب على الرفع |

## 🦺 3. معدات الحماية الشخصية (PPE)
* **حماية الرأس:** الخوذة (تستخدم في: الإنشاءات، المواقع المكشوفة).
* **حماية العين والوجه:** النظارات الواقية (تستخدم عند: اللحام، التعامل مع الكيماويات).
* **حماية التنفس:** الأقنعة والكمامات (تستخدم في: الغبار، الغازات السامة).

## 🧯 4. إجراءات الطوارئ والحرائق
* **أنواع الحرائق وطفاياتها:**
  * **الفئة A (مواد صلبة كالورق والخشب):** طفاية الماء أو البودرة.
  * **الفئة B (سوائل قابلة للاشتعال):** طفاية الرغوة أو ثاني أكسيد الكربون ($CO_2$).
  * **الفئة C (أجهزة كهربائية):** طفاية ثاني أكسيد الكربون ($CO_2$).
* **خطة الإخلاء:** (نقاط التجمع، إشارات الخروج).
